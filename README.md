# CloudAI

CloudAI 是一个纯前端现代化 AI 创作平台官网首页示例，使用 HTML、CSS 和 JavaScript 构建，无需任何后端服务。

## 页面内容

- Logo：CloudAI
- Hero 区域
- AI 文案工具介绍
- AI 图片工具介绍
- 功能特点
- 联系我们

## 项目结构

```text
.
├── index.html
├── styles.css
├── script.js
└── README.md
```

## 本地运行

直接在浏览器中打开 `index.html`，或使用任意静态文件服务器预览：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 技术说明

- 所有样式位于 `styles.css`
- 所有脚本位于 `script.js`
- 使用相对路径引入本地资源
- 支持桌面端和移动端自适应布局
